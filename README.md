# Cook Timer Mobile

An Expo-based mobile cooking timer with:

- recipe presets for common dishes
- a large animated countdown
- shimmer and steam motion while the timer is running
- start, pause, and reset controls

## Run locally

1. Install dependencies with `npm install`
2. Start the app with `npm run start`
3. Open it in Expo Go or a simulator
